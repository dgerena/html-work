Logitech unifying receivers

Logitech has a history of creating inexpensive yet capable pc peripherals for years. There newest unifying receiver takes that one step further. The receiver allows multiple devices that would normally require individual receivers to link using a single receiver.

devices that use the receivers.

  • logitech 2012 wireless keyboard.
  • logitech m570 wireless trackball.
  • logitech wireless stereo headset.

current limitations of unifying receivers.

  1. Current series supports only a single device.
    Any device once set to a receiver must be reassigned to be used with another receiver. A device may only be synced to one receiver at any one time.
  2. The receiver has a limiting distance of 15 feet direct sight.
    This is further inhibited by obstacles brining most users to be next to said devices.
  3. Currently only supports one type of device on any one receiver.

In other words

  • One keyboard
  • One Mouse
  • One headset
